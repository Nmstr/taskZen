### Move Relative

The `moveRelative` module moves the mouse to a position relative to the current position. It takes 2 arguments. `x` and `y`, which are the amounts units the mouse should be moved.

Parameters:
  - x: int
  - y: int

Example usage:
```
taskZen.moveRelative(100, 100)
```
